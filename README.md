## 平台概述 (Platform Overview)

**Flow-Wise** 是一套专为解决依赖移动人力和车辆的**现场服务行业 (Field Service) 通用痛点**而打造的一体化、信息化的云平台解决方案。我们深刻理解传统现场服务业中存在的调度效率低下、服务过程不透明、管理成本高昂、客户满意度难以把控等核心挑战。

本平台致力于将**工单管理 (Work Order)、智能调度 (Dispatch)、移动作业 (Mobility)、GIS追踪与大数据分析**等先进技术深度融合，为企业提供从客户下单、资源调度、现场执行、财务结算到运营决策的全链路数字化引擎。

我们的目标是帮助任何拥有“移动服务单元”（无论是人还是车）的企业，实现**运营成本的显著降低、服务效率的倍数提升、以及管理决策的数字化转型**。

## 核心理念：从垂直行业到通用平台

我们的平台之所以具备强大的跨行业适用性，是因为我们抽象并固化了现场服务的核心业务流程：

**工单生成 (Task Creation) -> 智能调度 (Intelligent Dispatch) -> 现场执行 (Field Execution) -> 过程追踪 (Process & Asset Tracking) -> 服务确认 (Service Confirmation) -> 结算闭环 (Billing & Analytics)**

通过对这个模型的深度数字化，我们可以通过简单的配置和UI调整，将平台快速适配到各行各业。

## 适用行业 (Applicable Industries)

本平台内核具备极高的灵活性和可配置性，可快速定制并应用于以下众多行业：

*   ### **道路救援**
    *   **场景:** 4S店、保险公司、汽车俱乐部、专业救援公司的拖车、搭电、换胎等救援任务调度。
    *   **实现:** 平台的原始应用场景，功能完整覆盖，经过深度商业验证。

*   ### **物流与配送**
    *   **场景:** 同城货运、快递末端派送、生鲜冷链、大件商品送装一体、即时配送（外卖、商超）。
    *   **定制:** `救援` -> `配送`；`拖车` -> `货车/配送员`；`救援完成` -> `签收`。

*   ### **设备安装与维保**
    *   **场景:** 电梯、空调、网络设备、工业机械、安防监控、充电桩的安装、巡检和上门维修。
    *   **定制:** `工单` -> `维保单`；`技师` -> `工程师`；`备件管理`。

*   ### **家政与上门服务**
    *   **场景:** 家庭保洁、家电维修、管道疏通、开锁换锁、保姆月嫂等服务的派单与管理。
    *   **定制:** `车辆` -> `服务人员`；`任务状态` -> `服务中/已完成`。

*   ### **安保巡逻**
    *   **场景:** 园区/社区的保安巡逻路线规划、打点签到、突发事件的就近安保力量调度。
    *   **定制:** `任务` -> `巡逻路线/紧急事件`；`轨迹回放` -> `巡逻复盘`。

*   ### **租赁与资产管理**
    *   **场景:** 共享汽车/单车的调度维护、工程机械租赁的设备调度与状态监控。
    *   **定制:** `车辆状态` -> `可用/租用中/维护中`；`电子围栏` -> `运营区域管理`。

*   ### **移动销售与医疗**
    *   **场景:** 移动售货车、外勤销售人员客户拜访路线规划、上门核酸采样/护理服务调度。
    *   **定制:** `任务` -> `拜访计划/采样任务`；`客户管理`。

## 技术架构与亮点 (Technical Architecture & Highlights)

我们采用成熟、稳健的微服务架构，确保平台的高性能、高可用性和未来的高可扩展性。

*   **后端核心 (Backend Core):**
    *   **技术栈:** **Java (Spring Boot + MyBatis)** + **Spring Cloud**，业界公认的企业级黄金组合，确保系统健壮、稳定、易于维护和扩展。
    *   **数据存储:** **MySQL** (核心业务数据) + **Redis** (高性能缓存/消息队列)，实现数据读写与存储成本的最优平衡。
*   **前端应用 (Frontend Applications):**
    *   **PC管理端:** **Vue 3 + Vite + Element Plus**，为调度员、财务和管理者提供功能强大、交互优秀的现代化Web操作后台。
    *   **移动作业端:** 支持 **App/小程序** 形式，为一线现场人员提供任务接收、导航、拍照上传、电子签名等功能，实现PC与移动端的无缝联动。
*   **云原生与DevOps:**
    *   全面拥抱 **Docker** 容器化，实现标准化、可移植的部署。
    *   结合 **Jenkins** 和 **Kubernetes (K8s)**，构建成熟的 **CI/CD (持续集成/持续部署)** 流程，实现高效的自动化运维。

## 完备的核心功能 (Comprehensive Core Features)

*   **智能调度与指挥中心 (Dispatch & Command Center)**
    *   **可视化调度:** 在GIS地图上实时展示所有服务单元（人/车）的位置、状态（空闲、任务中等），资源分布一目了然。
    *   **智能派单:** 系统根据任务地点、实时路况、资源技能、繁忙程度等多种策略，自动推荐最优派单方案，支持一键派单与多人抢单模式。
    *   **BI指挥大屏:** 实时动态展示订单量、完成率、响应时长、客户/员工排行等核心KPI，辅助管理层进行数据化决策。

*   **标准化现场服务流程 (Standardized Field Workflow)**
    *   **全程状态追踪:** 从接单、出发、到达、作业开始到服务完成，全过程节点实时上报，状态透明可控。
    *   **移动作业与数据采集:** 移动端支持现场拍照/录像上传、自定义表单填写、物料消耗登记、客户电子签名确认，所有凭证云端存储，有据可查。
    *   **任务轨迹回放:** 自动记录并回放每一次任务的移动轨迹，便于事后复盘、成本核算与纠纷处理。

*   **自动化财务管理 (Automated Financials)**
    *   **灵活计费引擎:** 支持按里程、按工时、按项目、阶梯计价等多种计费模型，自动核算订单费用。
    *   **自动对账结算:** 可根据与客户预设的结算规则，一键生成对账单，支持分批结算，极大解放财务人力。
    *   **薪酬提成核算:** 根据服务人员的完成量和预设提成方案，自动计算薪资，一键导出报表。

*   **资源与资产管理 (Resource & Asset Management)**
    *   **服务单元管理:** 建立完整的人员/车辆档案，管理资质、技能、联系方式等信息。
    *   **设备维保预警:** 系统根据车辆行驶里程或设备运行时间，自动触发保养、年检、校准等提醒，确保资源安全合规。
